# Tomer DayBoard

אפליקציית לוח יום ב-React/Vite עם:

- משימות עם סטטוס: לא בוצע / בוצע + שדה טקסט חופשי לאיפה המשימה עומדת
- תזונה יומית עם חישוב אוטומטי לפי מאכל + גרמים, כולל מאגר מורחב של חטיפים ושוקולדים
- ספר מוצרים לשמירת מאכלים קבועים
- מים עם יעד יומי
- היסטוריית תזונה ושתייה לפי ימים
- משקל והיסטוריית שקילות
- שמירה מקומית בדפדפן בעזרת localStorage
- PWA בסיסי להוספה למסך הבית בטלפון

## הרצה מקומית

```bash
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```

## העלאה ל-Vercel

- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

## הערה על חישוב התזונה

החישוב האוטומטי משתמש במאגר הערכות מקומי בתוך הקוד לפי 100 גרם. זה נועד לתת הערכה מהירה. תמיד אפשר לערוך את הקלוריות/חלבון/שומן ידנית לפני שמירה.
